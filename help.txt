usage: coindust <command> [<args>]

$ coindust new [<name>]
creates new bitcoin address and saves the public and private key

$ coindust import [<name>]
stores an existing bitcoin address using a private key in WIF form

$ coindust balance [ [<bitcoin address>] | [--wallet <bitcoind wallet dump file>] ]
check the balance of an address or a bitcoind wallet dump file

$ coindust tx --in <address> --out <address> --amount <btc> --wallet <filename>
build and display bitcoin transaction. Safe to use - DOES NOT SUBMIT TRANSACTION.
